== Description ==

Very simple plugin only add news Categories, news Tags and trash to WordPress Media Library.

== Installation ==

1. Upload the entire `lab-media-categories-tags` folder to your `/wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3.Just activate and enjoy.
